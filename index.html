📊 SQL Project – Analyzing Social Impact Movies

🎯 Objective:

This project uses SQL to store and analyze real Indian movies that portray disability or social issues.

🗃️ Tables Used:

  • Movies(MovieID, Title, Language, YearReleased)
  • Ratings(RatingID, MovieID, Rating)

🧪 Sample Data:

  • Deiva Thirumagal (Tamil, 2011)
  • Ambili (Malayalam, 2019)
  • Taare Zameen Par (Hindi, 2007)
  • Black (Hindi, 2005)
  • Margarita with a Straw (2014)
  • My Name is Khan (2010)
  • Barfi! (2012)
  • Paa (2009)

🎯 Ratings:

  • Deiva Thirumagal: 9, 8
  • Taare Zameen Par: 10, 10
  • Ambili: 8
  • Barfi!: 9
  • Black: 9
  • Others: 7–9

💻 SQL Queries:

1. Show all movies

SELECT * FROM Movies;

2. Show all ratings

SELECT * FROM Ratings;

3. Average rating per movie

SELECT Movies.Title, AVG(Ratings.Rating) AS AverageRating
FROM Movies
JOIN Ratings ON Movies.MovieID = Ratings.MovieID
GROUP BY Movies.Title;

4. Movies with rating above 8

SELECT DISTINCT Movies.Title
FROM Movies
JOIN Ratings ON Movies.MovieID = Ratings.MovieID
WHERE Ratings.Rating > 8;

📈 Results:

  • Highest rated: Taare Zameen Par (10), Barfi!, My Name is Khan
  • Multi-ratings: Deiva Thirumagal – 8.5 avg
  • Lowest rating: Paa – 7

💡 What I Learned:

This project helped me practice SQL queries like SELECT, JOIN, WHERE, and GROUP BY using real-life data. I also learned how to organize data in tables and find meaningful insights.
